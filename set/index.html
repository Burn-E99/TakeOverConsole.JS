Innovators Debug Enabler

Some issues are difficult to reproduce, or only happen on certain devices, so we created a simple debugger to allow us to debug devices remotely and immediately (no reload needed!).

This tool enables the custom-built TACO_TRACKER (TACO is a shortened name for TakeOverConsole), which is a very simple, small tool (1.68 KB non-minified!) that takes all the JavaScript console outputs from an enabled device and sends it to TACO's own log, so that developers at Innovators Robotics can attempt to identify the issue. This tool does NOT collect any information outside of what the enabled website outputs, in other words it does NOT hunt down personal information to send back.

Enabling and disabling TACO_TRACKER

To enable TACO_TRACKER, fill out the form below. Enter your team number (enter 9999 if you are not on a team), and a unique identifying number for this device (if you were provided with a unique identifying number, use that), and click "Save".

To disable TACO_TRACKER, press the button below. Please disable the tracking after you have finished reproducing your issue. The logs will still be saved, but will not get polluted with extra information
